# Dumopro 재배포

## 1. [서버] 비우기

```bash
cd ~/Dumopro 2>/dev/null && docker compose down -v
rm -rf ~/Dumopro ~/projects/dumopro
docker rmi -f dumopro-dumopro-api:latest dumopro-dumopro-poller:latest redis:7-alpine
mkdir -p ~/projects
```

## 2. [내 PC] 전송

```powershell
scp -r -P 50022 "C:\output\dumopro" daim@10.5.20.160:~/projects/
```

## 3. [서버] 설치

```bash
cd ~/projects/dumopro
chmod +x INSTALL.sh
bash INSTALL.sh
```

## 4. [서버] 실행 확인

```bash
docker compose -f ~/Dumopro/docker-compose.yml ps
curl -s http://localhost:9105/api/health
```
