# SocketDaim 재배포 (DB / 라벨 / 영상 보존)

## 1. [서버] 비우기 (볼륨 안 건드림)

```bash
cd ~/SocketDaim 2>/dev/null && docker compose down
rm -rf ~/projects/socket_daim
docker rmi -f socketdaim-ingestion-gw:latest socketdaim-admin-ui:latest \
              socketdaim-cleaner:latest socketdaim-egress-gw:latest postgres:16
mkdir -p ~/projects
```

## 2. [내 PC] 전송

```powershell
scp -r -P 50022 "C:\output\socket_daim" daim@10.5.20.160:~/projects/
```

## 3. [서버] 설치

```bash
cd ~/projects/socket_daim
chmod +x INSTALL.sh
bash INSTALL.sh
```

## 4. [서버] 실행 확인

```bash
docker compose -f ~/SocketDaim/docker-compose.yml ps
docker exec sd-postgres psql -U postgres -d gateway_db -c "
SELECT (SELECT count(*) FROM dust_inspection) AS dust,
       (SELECT count(*) FROM cctv_frame)      AS cctv,
       (SELECT count(*) FROM waypoint_label)  AS labels;"
```
